# Imprint

Analyze any website or GitHub repository for AI-generated design and code patterns. Get a score from 0 to 100.

---

## Stack

Next.js 15 · TypeScript · Playwright · Cheerio · Tailwind CSS v4

---

## Running locally

```bash
git clone https://github.com/Mounil2005/imprint.git
cd imprint
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
